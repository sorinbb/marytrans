Year field
==========================

Description
-----------
This module provides a field type for year without using date module.


Installation
------------
1. Enable the module.

Usage
-----
1. For new field select "Year" field type
2. Configure default settings for field.
 - Start year (You can use default year integer value like "2001" or use strtotime strings)
 - End year (You can use default year integer value like "2001" or use strtotime strings)
 - Default value
3. Choose "Academic year" if you need show year like "2017 - 2018". (This is just widget. in database will be stored only 2017 value!)


Work with
-----

 - Views exposed filters (select list)
 - DDF Entity Reference (https://www.drupal.org/project/ddf)
 - Views Reference Filter (https://www.drupal.org/project/entityreference_filter)
